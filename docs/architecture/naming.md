# Brand & Naming

## Product Name

**Pictovap** is the public product name. It is used in:
- README and documentation
- GitHub repository name
- PyPI package name
- User-facing descriptions

## Internal Package Name

The internal Python package is named `pictova` for backward compatibility.

| Context | Name | Reason |
|---------|------|--------|
| Product name | Pictovap | Public brand identity |
| Python package | `pictova` | Backward compatibility with existing imports |
| CLI command | `pictova` | Backward compatibility with existing scripts |
| Repository | `Pictovap` | GitHub repository name |
| PyPI package | `pictovap` | Published package name |

## Etymology

The name derives from Latin *pictus* (past participle of *pingere*, to paint or depict) combined with the suffix *-vap*, suggesting visual automation for publishers.

## Future Migration

A future major version may rename the internal package from `pictova` to `pictovap` for consistency. This will be a breaking change and will be documented in the changelog with migration instructions.
