# Pictovap 0.7.12

Pictovap 0.7.12 lowers the cost of real external validation and static-site
CMS contributions.

## Added

- A runnable standalone Hugo CMS adapter reference at
  `examples/adapters/pictovap-hugo`.
- `pictovap feedback --format markdown` for GitHub issue-ready validation
  reports that exclude article text, private paths, image URLs, profile names,
  and credentials.
- A dedicated external validation issue template.

## Fixed

- GitHub Codespaces contributor setup now installs the full local gate
  dependency set.
- Adapter and renderer example packages use the current public runtime floor.
- Adoption and issue-plan drafts use the canonical `pictovap plan` command.
