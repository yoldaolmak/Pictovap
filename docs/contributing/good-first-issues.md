# Good First Issues

Contribution status changes faster than a documentation page can be released.
The canonical sources are therefore:

- [July 2026 Adapter Sprint](adapter-sprint.md) for the current focused sprint.
- [WordPress Gutenberg Mini Sprint](wordpress-gutenberg-minisprint.md) for
  small fixture, test, and report contributions around image placement.
- [Open `good first issue` work](https://github.com/yoldaolmak/Pictovap/issues?q=is%3Aissue%20is%3Aopen%20label%3A%22good%20first%20issue%22) for all beginner-scoped issues.
- [Open `help wanted` work](https://github.com/yoldaolmak/Pictovap/issues?q=is%3Aissue%20is%3Aopen%20label%3A%22help%20wanted%22) for larger contributions.

Claim work by commenting on the live issue with your intended approach and test
plan. Do not use copied issue lists in old release notes or external posts as a
status source.

See [CONTRIBUTING.md](../../CONTRIBUTING.md) for the complete pull-request process.
