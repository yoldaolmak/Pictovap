# External HTML Review Renderer

This is a complete, independently installable Pictovap plugin package. It is
not imported by Pictovap core. Installation exposes its renderer through the
standard `pictovap.report_renderers` entry-point group.

## Run It From This Checkout

```bash
cd examples/external-renderer-package
python -m venv .venv
source .venv/bin/activate
pip install -e ../..
pip install -e ".[test]"
pytest

pictovap plugins --kind renderer
pictovap adapter check --kind renderer --name external-html-review

pictovap plan \
  --article ../articles/travel-guide.md \
  --profile ../profiles/sample-publisher.yaml \
  --output plan.json
pictovap report --plan plan.json --output review.html --renderer external-html-review
```

`review.html` is generated by this distribution, not by a built-in renderer.
The example has no credentials and no network dependency.

## Package Shape

- `pyproject.toml` declares the distribution and the entry point.
- `src/pictovap_external_html_review/` contains the adapter implementation.
- `tests/test_renderer.py` imports Pictovap's public contract helper.

To publish a real renderer, rename the distribution and entry-point name,
replace the presentation logic, keep the contract test, and declare the
compatible Pictovap version range.
